📋 เนื้อหาของคู่มือ (11 บท):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 1️⃣  บทนำและความรู้พื้นฐาน
      - ระบบ Classroom Finance คืออะไร
      - ส่วนประกอบและประโยชน์

 2️⃣  การติดตั้งและเตรียมระบบ
      - ตรวจ Node.js, npm
      - โคลนโครงการ
      - ติดตั้ง Dependencies

 3️⃣  การตั้งค่า Supabase (ฐานข้อมูล)
      - สร้าง Supabase Project
      - ดึง Credentials
      - รัน Migrations

 4️⃣  การตั้งค่า LINE Bot
      - สร้าง LINE Channel
      - ตั้งค่า Webhook
      - ดึง LINE Credentials

 5️⃣  การใช้งานระบบจริง (First Run)
      - รัน Dev Server
      - เพิ่มข้อมูลเริ่มต้น
      - ตั้คค่า Settings

 6️⃣  การใช้งานแต่ละหน้า
      - Dashboard (หน้าหลัก)
      - Transactions (บันทึกรายการ)
      - Schedule (กำหนดการ)
      - Students (นักเรียน)

 7️⃣  หน้าตั้งค่า (Settings)
      - ตั้งค่าลับ (Secrets)
      - บัญชีรับเงิน (Receiver)
      - เมนู LINE (Rich Menu)
      - ข้อความ LINE

 8️⃣  QR Code และวิธีชำระเงิน
      - ประเภทการชำระ
      - ตั้งค่า QR
      - วิธี Generate

 9️⃣  การตรวจสลิป (Slip Checking)
      - ขั้นตอนการตรวจ
      - ประเภทผลลัพธ์
      - วิธีอนุมัติ/ปฏิเสธ

 🔟  คำถามที่พบบ่อย (FAQ)
      - 17 คำถามพร้อมคำตอบ
      - ครอบคลุมทุกด้าน

 1️⃣1️⃣  การแก้ไขปัญหา (Troubleshooting)
      - 13 ปัญหาทั่วไป
      - วิธีแก้ไขแต่ละข้อ
      - Checklist ส่งต่อ
