## 🚀 วิธีสร้าง PDF (เลือกวิธีใดวิธีหนึ่ง)

### ✨ วิธีที่ 1: ใช้เบราว์เซอร์ (ง่ายที่สุด! ⭐)

**ขั้นตอน:**
1. เปิด `คู่มือการเริ่มต้นใช้งาน_ฉบับสมบูรณ์.html` ในเบราว์เซอร์
2. กด `Cmd+P` (Mac) หรือ `Ctrl+P` (Windows)
3. เลือก "Save as PDF"
4. ตั้งชื่อ: `คู่มือการเริ่มต้นใช้งาน_Classroom_Finance`
5. คลิก "Save"

**เวลา:** 2 นาที | **ความยากง่าย:** ง่าย | **คุณภาพ:** ⭐⭐⭐⭐⭐

---

### 🖥️ วิธีที่ 2: ใช้ Shell Script (macOS/Linux)

```bash
cd /path/to/classroom-finance
bash html-to-pdf.sh
```

**เวลา:** 30 วินาที | **ความยากง่าย:** ปานกลาง

---

### 🐍 วิธีที่ 3: ใช้ Python

#### 3a. ใช้ Virtual Environment (ปลอดภัยที่สุด)
```bash
# สร้าง virtual environment
python3 -m venv pdf_env
source pdf_env/bin/activate  # macOS/Linux
# หรือ
pdf_env\Scripts\activate  # Windows

# ติดตั้ง weasyprint
pip install weasyprint

# แปลง
python3 convert_to_pdf.py
```

#### 3b. ใช้ Homebrew (macOS)
```bash
# ติดตั้ง wkhtmltopdf
brew install wkhtmltopdf

# ใช้เบราว์เซอร์แปลง (วิธีที่ 1)
```

**เวลา:** 5 นาที | **ความยากง่าย:** ปานกลาง-ยาก

---

### 📦 วิธีที่ 4: ใช้ Node.js

```bash
# ติดตั้ง puppeteer
npm install puppeteer

# แปลง
node html-to-pdf.js
```

**เวลา:** 3 นาที | **ความยากง่าย:** ปานกลาง
