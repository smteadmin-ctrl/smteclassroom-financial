## 🎯 เนื้อหาคู่มือ

### 11 บท รวมทั้งหมด

#### **บท 1: บทนำและความรู้พื้นฐาน**
- ระบบ Classroom Finance คืออะไร
- ส่วนประกอบของระบบ (เว็บ, LINE Bot, Supabase)
- ประโยชน์ของระบบ
- ความรู้ที่ต้องการ (สำหรับผู้ใช้มือใหม่)

#### **บท 2: การติดตั้งและเตรียมระบบ**
- ความต้องการทางด้านเทคนิค (Node.js, npm, Git)
- ขั้นตอนตรวจสอบ Node.js และ npm
- โคลนโครงการจาก GitHub
- ติดตั้ง Dependencies
- ตั้งค่า Environment Variables

#### **บท 3: การตั้งค่า Supabase (ฐานข้อมูล)**
- ขั้นตอนที่ 1: สร้างบัญชี Supabase
- ขั้นตอนที่ 2: สร้าง Supabase Project
- ขั้นตอนที่ 3: ดึง API Credentials
- ขั้นตอนที่ 4: อัปเดต .env.local
- ขั้นตอนที่ 5: รัน Database Migrations

#### **บท 4: การตั้งค่า LINE Bot**
- ขั้นตอนที่ 1: สร้าง LINE Channel
- ขั้นตอนที่ 2: ดึง Credentials จาก LINE
- ขั้นตอนที่ 3: ตั้งค่า Webhook URL
- ขั้นตอนที่ 4: อัปเดต .env.local
- ข้อเตือนและหมายเหตุสำคัญ

#### **บท 5: การใช้งานระบบจริง (First Run)**
- ขั้นตอนที่ 1: รัน Development Server
- ขั้นตอนที่ 2: ทำความรู้จักกับ Interface
- ขั้นตอนที่ 3: เพิ่มข้อมูลเริ่มต้น (เพิ่มนักเรียน)
- ขั้นตอนที่ 4: ตั้คค่า Settings

#### **บท 6: การใช้งานแต่ละหน้า**
- **🏠 หน้า Dashboard**
  - ส่วนประกอบของ Dashboard
  - ข้อมูลในแต่ละการ์ด
  - วิธีใช้ Dashboard

- **💳 หน้า Transactions**
  - ประเภทธุรกรรม (Income, Expense, Transfer)
  - วิธีเพิ่มธุรกรรม
  - วิธีแก้ไขธุรกรรม
  - วิธีค้นหาและกรอง

- **📅 หน้า Schedule**
  - ข้อมูลกำหนดการ
  - วิธีเพิ่มกำหนดการ
  - วิธีดูรายละเอียด
  - การจัดการนักเรียน

- **👥 หน้า Students**
  - ข้อมูลนักเรียน
  - วิธีเพิ่มนักเรียน
  - อัพโหลดรูปโปรไฟล์
  - ดูประวัติการชำระ

#### **บท 7: หน้าตั้งค่า (Settings)**
- หมวด 1: Secrets (ข้อมูลลับ)
  - Supabase URL & Key
  - LINE Channel ID, Secret, Access Token
  - EasySlip API Key
  - Vercel Blob Token

- หมวด 2: Receiver (บัญชีรับเงิน)
  - PromptPay ID
  - ข้อมูลบัญชีธนาคาร
  - ข้อมูล TrueMoney

- หมวด 3: Rich Menu (เมนูปุ่ม LINE)
  - ตั้งค่าปุ่ม 4-6 ปุ่ม
  - ชื่อและ Action Text
  - วิธี Setup Rich Menu

- หมวด 4: Messages (ข้อความ LINE)
  - ปรับแต่งข้อความ 15+ รายการ
  - ใช้ตัวแปร {{studentName}} เป็นต้น

#### **บท 8: QR Code และวิธีชำระเงิน**
- ประเภทการชำระเงิน 3 วิธี
- ตั้งค่า QR ธนาคาร
- ตั้งค่า QR TrueMoney
- วิธี Generate QR ตอนใช้งาน
- ตรวจสอบ QR

#### **บท 9: การตรวจสลิป (Slip Checking)**
- ขั้นตอนการตรวจสลิป (9 ขั้นตอน)
- ประเภทของผลการตรวจ (Auto Approve, Pending, Auto Reject)
- ตั้งค่า Slip Checker
- วิธีตรวจและอนุมัติสลิป
- เคล็ดลับการตรวจ

#### **บท 10: คำถามที่พบบ่อย (FAQ)**
- 🤔 คำถามทั่วไป (5 ข้อ)
- 🔧 คำถามทางเทคนิค (5 ข้อ)
- 💳 คำถามเกี่ยวกับการชำระเงิน (3 ข้อ)
- 📱 คำถามเกี่ยวกับ LINE (4 ข้อ)
- รวมทั้งหมด 17 คำถาม

#### **บท 11: การแก้ไขปัญหา (Troubleshooting)**
- ⚠️ ข้อผิดพลาดทั่วไป (7 ประการ)
- 🚀 Deploy ปัญหา (3 ประการ)
- 📊 Supabase ปัญหา (3 ประการ)
- 💡 คำแนะนำ
- Checklist ส่งต่อรุ่นน้อง
